# AltGesture 权限说明

AltGesture 按最小授权运行：默认只请求核心功能需要的权限；屏幕录制和自动化都不是默认必需项。

## 权限清单

| macOS 权限 | 是否必需 | 用途 | 不授权的影响 |
| --- | --- | --- | --- |
| 辅助功能 | 核心功能需要 | 聚焦、恢复、最小化、移动窗口；右键手势事件拦截也依赖它稳定工作 | 窗口切换、最小化、布局和右键手势会不稳定或不可用 |
| 输入监控 | 核心功能需要 | 监听右键鼠标手势、双击右键、右键滚轮切换、鼠标组合键、连续三击右键、双击 Option 和 Option 释放事件 | 右键手势、双击右键最小化、右键滚轮切换、三击右键黑屏和双击 Option 不可用；Option-Tab 释放确认会降级 |
| 屏幕录制 | 可选 | 只用于生成本地窗口缩略图 | 关闭缩略图即可不授权；窗口列表会使用应用图标 |
| 自动化 | 条件需要 | 仅当某个自定义右键手势配置为通过 System Events 发送快捷键时使用 | 这些特定手势无法触发目标应用的全局快捷键；其他功能不受影响 |

## 最小授权建议

如果只想保持核心体验，授权：

- 辅助功能
- 输入监控

不要授权或不要开启：

- 屏幕录制：保持菜单里的“显示窗口缩略图”关闭。
- 自动化：避免使用配置里带 `"delivery": "systemEvents"` 且没有 `"windowAction"` 的自定义右键手势。内置窗口布局不需要自动化权限。

## 替代 Magnet

AltGesture 已内置常用窗口布局能力：

- 左半屏、右半屏、上半屏、下半屏
- 左上、右上、左下、右下四角
- 居中、铺满、还原

从旧配置迁移来的 `Control + Option + 方向键` 手势会自动改成本机窗口动作，不再需要安装 Magnet。

## 鼠标窗口切换

按住右键并滚动滚轮会打开同一个窗口切换器：向下滚动前进，向上滚动反向，松开右键确认。这个功能复用输入监控识别右键和滚轮，不需要新的权限。

## 隐私黑屏

连续三击右键会用黑色遮罩覆盖所有屏幕并隐藏鼠标指针，再次三击右键恢复。这个功能只复用输入监控识别右键点击，并用本地窗口覆盖屏幕；它不读取屏幕内容，不需要屏幕录制权限。

## 本地数据

- 日志：`~/Library/Logs/AltGesture.log`
- 右键手势配置：`~/Library/Application Support/AltGesture/right-gestures.json`
- 开机启动：`~/Library/LaunchAgents/local.alt-gesture.login.plist`

AltGesture 不联网、不上传数据、无遥测。
